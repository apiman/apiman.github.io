Getting Started

One easy way to get started with apiman is to simply download WildFly 8 and the apiman overlay. Unpack them both into the same location and you're off and running. You'll obviously also need to have Java installed - we currently recommend at least version 7.

Download

  • WildFly 8.1.0.Final
  • apiman 1.0.0.Alpha4 overlay for WildFly 8.1

Install

  1. Unpack the WildFly 8.1 zip
  2. Unpack the apiman 1.0.0.Alpha4 WildFly 8 overlay zip

Run

  1. Start WildFly 8 using the "standalone-apiman.xml" configuration
  2. Point your browser at the apiman UI and login with admin/admin123!

Configure

  1. Navigate to the Role Management section of the apiman UI
  2. Add whatever roles you need, but at a minimum we recommend adding a Organization Owner role with *all* permissions
  3. Navigate to the Policy Definitions section of the apiman UI
  4. Import apiman policy definition data that comes with the apiman overlay distribution
    The policy definition data is here: wildfly-8.1.0.Final/apiman/data/all-policyDefs.json

Or simply try this...

mkdir ~/apiman
cd ~/apiman
wget http://download.jboss.org/wildfly/8.1.0.Final/wildfly-8.1.0.Final.zip
wget http://downloads.jboss.org/overlord/apiman/1.0.0.Alpha4/apiman-distro-wildfly8-1.0.0.Alpha4-overlay.zip
unzip wildfly-8.1.0.Final.zip
unzip apiman-distro-wildfly8-1.0.0.Alpha4-overlay.zip
cd wildfly-8.1.0.Final
./bin/standalone.sh -c standalone-apiman.xml

Getting Started (Docker)

If you prefer to get started by using the apiman docker images, that's cool too! Skip all the steps above and follow the instructions in this section instead.

Download

  • Use docker to pull the "apiman on WildFly 8" docker image: apiman/on-wildfly8

Run

  • User docker to run the "apiman on WildFly 8" docker image

Configure

  1. Navigate to the Role Management section of the apiman UI
  2. Add whatever roles you need, but at a minimum we recommend adding a Organization Owner role with *all* permissions
  3. Download the apiman policy definition data
  4. Navigate to the Policy Definitions section of the apiman UI
  5. Import apiman policy definition data that comes with the apiman overlay distribution

Or simply try this...

echo "Grab the policy definition data for later use."
curl -L http://bit.ly/1v3oTfP -o all-policyDefs.json

echo "Download and run apiman on WildFly 8."
docker pull apiman/on-wildfly8
docker run -it -p 8080:8080 apiman/on-wildfly8
